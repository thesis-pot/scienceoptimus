# ScienceOptimus Website

A modern, responsive educational website for Class 9 to University Admission preparation (Science Group).

## Features
- Fully responsive design
- Course listing with filtering
- Student testimonials
- Contact form
- Mobile-friendly navigation
- Fast loading with optimized assets

## Pages
1. Homepage (`index.html`)
2. Courses (`courses.html`)
3. Free Resources (`resources.html`)
4. Blog (`blog.html`)
5. Contact (`contact.html`)

## Setup Instructions

### 1. Host on GitHub Pages
1. Create a new GitHub repository
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select "main" branch as source
5. Your site will be live at: `https://username.github.io/repository-name`

### 2. Customize Content
- Edit text in HTML files
- Replace images in `/images` folder
- Update colors in CSS variables
- Modify contact information in footer

### 3. Add More Features
- For payment integration: Add Stripe/Razorpay
- For user accounts: Use Firebase/Auth0
- For backend: Use Node.js/Django
- For database: Use MongoDB/PostgreSQL
